Simple Substitution Cipher

Simple Substitution Cipher, being an improvement to the Caesar Cipher, it follows the method of shifting the alphabets by some number using some permutation of the letters in alphabet which is nothing but a random jumbled set of alphabets.

To implement this, both the sender and the receiver may choose any one of these possible permutation as a ciphertext alphabet. The sender and the receiver decide on a randomly selected permutation of the letters of the alphabet. This permutation is the secret key of the scheme.
